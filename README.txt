The user of this app can:
- Write and store an arbitrary number of text-based notes.
- Go back and edit notes
- Delete notes from list
- Attach pictures to notes.
- Email notes.

Each note contains:
- Title
- Body of text
- Date and time of creation
- Picture (optional)
